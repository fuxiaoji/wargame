# V4 交叉评估报告

## 对战矩阵（德军胜率, 各10局）

| | V1英 | V2英 | V3英 | V4英 |
|---|---|---|---|---|
| V1德 | - | - | - | **70%** |
| V2德 | - | - | - | **60%** |
| V3德 | - | - | - | **70%** |
| V4德 | **50%** | **90%** | **80%** | **50%** |

> 注：样本仅 10 局/组，有统计波动（V4德 vs V4英 两次测试得 50%/60%）。

## 关键发现

### 1. V4 英军代际进步最大

V4德对 V2英 90% → 对 V4英 50%。V2/V3 英军没有 Patrol 策略，被 V4 德军（w1=8 冲港）轻松突破。V4 英军加入 Patrol 后，德军胜率从 90% 骤降到 50%。

### 2. 消融实验确认 Patrol +18%

在训练种群内 400 对测试中，有 Patrol 的英军比没有的强 ~18 个百分点。这个提升在跨代际对比中同样体现。

### 3. V1/V3 德军仍有威胁

V4 英军对 V1/V3 德军只有 30% 胜率。V1/V3 德军的 w1 低（2.7-3.3），策略保守（贪心打工），反而不容易被 Patrol 拦截——它们不走直线冲港，而是绕路。

### 4. V2/V3 英军是最弱基准

V4 德军对它们有 80-90% 胜率。V2 训练用 MAP-Elites 但无 Patrol，V3 训练在 bug 环境下收敛到极端特化——两者对外都脆弱。

## Bug 修复对代际对比的影响

本评估使用修复版引擎（正确 BFS、blockHexes 移除、迷雾修正）。关键影响：

1. **V1/V2/V3 训练时用的都是 bug 版引擎**：它们的权重在 bug 环境下进化，在修复版引擎下表现不同
2. **V3 最明显**：在 bug 版里 50-50，修复版里被 V4 德 80% 碾压
3. **V4 是唯一在修复版引擎下训练的**：权重适应正确规则

## 结论

1. **Patrol 策略显著有效**（+18%），应该保留
2. **V4 英军是史上最强**，但仍挡不住保守型德军（V1/V3）
3. **修复版引擎训练产生更鲁棒的 AI**
4. **下一代方向**：给德军更多样化的反制策略（不只是冲港）
