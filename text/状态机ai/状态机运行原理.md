# 状态机 AI 运行原理 (2026-05-25 最新版)

## 1. 概述

基于**热力图 + 策略概率 + 加权随机**的启发式决策。29 个可调权重，双引擎（TS + C++）。

```
游戏状态 → 基础热力图 → 策略选择(概率) → 热力图修正 → score(-heat+noise) → 加权随机选动作
```

---

## 2. 热力图系统

### 2.1 数值约定

| 值 | 含义 |
|----|------|
| 负值 | **吸引**，AI 倾向前往 |
| 正值 | **排斥**，AI 倾向避开 |

移动得分：`score = -heatmap.get(hex) + noise`（±0.05~0.1）。热力越低得分越高。

### 2.2 德军基础热力图（两组件，所有策略共享）

- **英军位置**：所有算子（真船+伪装）本格 +2，邻格 +1。德军所见均为 `?`，不区分。
- **威胁范围**：英军周围 3 格 BFS +0.5。

### 2.3 英军双层热力图

**共享基图**（4 策略全可见）：德军可能位置扩散圈
- 从未目击 → A5/A6/B7 扩散，`radius = turn × 2`
- 已目击 → 最后位置扩散，`radius = turnsSinceSeen × 2`
- 发现后立刻重置中心+计数

**策略叠加**（各自独立）：
| 策略 | 叠加 |
|------|------|
| search | 船间排斥(+2) 强制分散 |
| hunt | 中心-6 梯度扩散 + 近处抱团(-1) + 同格swarm |
| patrol | 航路(-3) + 路口(-1) |
| defend | F7(-6) + 邻格(-3) |

### 2.4 防死锁

连续停同一格 ≥2 回合才产生排斥 `+(visits-1)×4`。动了就清零。

### 2.5 热力图传播（可选）

`propagated(hex) = heat[hex] + Σ heat[c] / (1+dist)²`，4 模式：off/full/neg/pos。

## 3. 德军 AI

### 3.1 四大策略

**Rush（冲港）**
```
rush = w1×(8-distF7)/8 + w2×HP/最大HP + w3×(1-britNearF7/5)
     - w4×(found?2:0) - vpNeed×1.5
```
VP 不足时扣分（`vpNeed = max(0, 6-vp)`）。热力图：F7 -10 + 邻格 -5 + 路径 D8/E7/F6/D6/C6/C7 递减吸引。

**Farm（破交）**
```
farm = w5×onRoute + w6×(1-vp/6) + w7×(!found?1:0) + w8×0.3
```
热力图：航路 `-(2+(6-vp)×0.5)`，VP 越少越强（VP=0→-5）。

**Hunt（猎杀）**
```
hunt = w9×孤立英军数 + w10×0.5 - w11×身边英军数(≤3格)
```
孤立=周围 3 格无其他英军。不区分真船/伪装。

**Hide（躲藏）**
```
hide = w12×found + w13×(HP<2?1:0) + w14×英军平均距离/8 - w15×(1-vp/6)
```
热力图：英军周围 3 格 +8 排斥。

### 3.2 船特定修正

| | Rush | Farm | Hunt | Hide |
|---|------|------|------|------|
| 俾斯麦 | **+2** | 0 | **-3** | +1 |
| 欧根 | -2 | +1 | **+3** | 0 |

### 3.3 策略选择

原始得分 → softmax(temperature) → 95%+5%均匀 → `weightedPick(原始得分, temperature)`。**用原始得分避免双重softmax**。

### 3.4 运输攻击

farm/hunt 必打；rush/hide 跳过。

## 4. 英军 AI

### 4.1 四大策略

**Search（搜索）**
```
search = s1×(!found?1:0) - s2×turnsSinceSeen - s3×vp_g
```
热力图叠加：船间排斥(+2) 分散覆盖。

**Hunt（追猎）**
```
hunt = h1×found + h2/(distToLast+1) + h3 - crowdPenalty
crowdPenalty = max(0, huntCount-5)×1.5
```
热力图：中心(-6) → 扩散梯度 `-(4-dist×0.6)` → 外围(+1 散开)。同格 swarm 全图 `-5/(1+dist)`。

**Patrol（护航）**
```
patrol = p1×onRoute + p2×inGermanRange + p3×(1-found×0.5)
```
热力图：航路(-3) + 路口 D7/C7/D6(-1)。

**Defend（防守）**
```
defend = d1×germanNearF7 + d2×(vp_g≥5?1:0) + d3×(1-shipsNearF7/5) + proactive
proactive = turn≥8:+1, vp≥3:+1, vp≥5:+2
```
热力图：F7(-6) + 邻格(-3)。

### 4.2 索敌

航空索敌优先冲港路线格(分 5) > A/B 列(分 3) > 其他(分 1)。

### 4.3 初始布置

固定舰船归位 → `getGermanReachableLabels` 算速2可达格（排除 A5/A6/B7+固定格）→ 真船→伪装→fallback。
