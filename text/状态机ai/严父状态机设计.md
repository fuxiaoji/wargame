## 德军 "严父"

### 固定路线
```
T1: 初设 B7
T1: 俾斯麦 → D8, 欧根 → D8
T2: 俾斯麦 → F7, 欧根 → E5
T3+: 俾斯麦 蹲 F7, 欧根 在 E5 袭击运输
```

### 欧根亲王逻辑
- 优先袭击运输（有运输攻击选项就执行）
- 如位置暴露且 VP 未领先：换航路（E4 / D5 / F4）
- 如受伤步数不足：沿路径一步步走（当前格→下一节点）

### 俾斯麦逻辑
- T1: B7→D8
- T2: D8→F7
- T3+: 蹲 F7 不动
- 如被击退：重新向 F7 移动

## 英军 "严父"

### 航路格
```
D2, D3, C3, C4, D5, E1, F4, E4, E5  (9格)
```

### 初设
- 真船（11艘）+ 伪装（4艘）= 15艘
- 尽可能均匀分配到 9 个航路格（每格 1-2 艘）
- 伪装贴在真船少的格

### 未发现德军时
- 每回合记录"上回合未覆盖的航路格"
- 优先向未覆盖格移动
- 保持每格至少 1 艘

### 发现德军后
- 所有船铺开到航路格
- 每格至少 1 艘，多余的去相邻航路格支援
- 战斗损失后：下回合重新均分

### 不动了
- 到位后不再移动（finish-phase）
