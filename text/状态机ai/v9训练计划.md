# V9 训练计划

## 背景

V1-V8 在 buggy 框架下训练（扩散梯度、turnsSinceSeen 按船计数、lastKnownGermanPos 不回退、英军系数硬编码、英军用德军权重）。V8 权重在新框架下德军胜率从默认 83% 暴跌到 37%，不可用。

V9 在修正后的框架下从头训练，全部 41 个权重（+5 个英军热力系数）。

## 已修复的 Bug（影响训练）

| Bug | 修复 | 对英军影响 |
|-----|------|----------|
| 扩散梯度线性衰减 | 改为半径内均匀 `-britDiffuseStr` | 德军可能区域正确覆盖 |
| turnsSinceSeen 按船计数 | 改为回合级 `state.turn - lastSeenTurn` | 扩散半径正确增长 |
| lastKnownGermanPos 不回退 | 从 `lastSightingTurn` 检测新目击 | 同格反复目击正确刷新 |
| hunt 梯度线性衰减 | 改为半径内均匀 `-britHuntCenter` | hunt 引力均匀 |
| C++ 英军 5 系数硬编码 | 接入 `w.britXxx` | 训练可优化这些权重 |
| C++ defend/hunt/search 公式 | 与 TS 同步 | 训练行为一致 |
| 扩散半径硬编码 2 | 改为 `germanSpeed`（受伤=1） | 受伤俾斯麦扩散减半 |
| 伪装只跟随一回合 | 改为持续跟随直到鉴定成功 | 英军持续获得精确位置 |
| F4 缺失航路 | 加入 patrol/farm 列表 | 非洲航路正确覆盖 |
| 英军使用德军权重 | britW 正确传入 | 前端对战正确 |

## V9 新增改进

| 改进 | 说明 |
|------|------|
| **I1 扩散置信度** | 高置信(刚目击)→hunt +3；低置信(全图)→patrol +2（不极端） |
| **I2 紧急 F7 防守** | 德军 1 回合可达 F7 且 VP 领先 → defend +10，F7&邻格引力翻倍 |
| **I3 VP 推断** | 未发现但 VP 涨 → 航线格追加 -3 引力（德军在刷运输） |
